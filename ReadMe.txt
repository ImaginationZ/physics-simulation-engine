这是我们的物理大作业。我们设计了一个3D程序模拟弹性刚体的碰撞。
所有的物体可以实现弹性碰撞，刚体可以实现旋转。碰撞可以实现越界检查。
物体的定义在物体名头文件中。系统的定义和维护物体运动的函数在control.h头文件中。

程序使用C++编写，图形界面部分使用OpenGL。程序可在windows与linux下编译运行。
在windows下运行可能会要求系统需要GLUT32.dll文件。若提示需要GLUT32.dll文件，请将文件夹中GLUT32.dll放入系统C盘windows\system32文件夹中。若仍然出现问题请与我们联系。

给出的实例程序为collision.exe。
在给出的这个实例中，我们模拟了小球注入杯子，实现最适堆积，并连同杯子下落并反弹的场景。
程序的默认输入文件为input.txt，请与exe文件放在同一文件夹下运行。

操作说明：
待球完全落下并堆积好之后，按“M”即可下落。


视角操作：
鼠标滚轮可控制缩放大小。
左键拖动可以改变视角，右键重置。

按下“Q”键或者“ESC”键可以退出。
所有的按键请在大写锁定下输入。
